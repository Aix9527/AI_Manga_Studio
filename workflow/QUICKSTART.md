## 🚀 快速使用指南

### 第 1 步：配置 API

1. 找到 `OpenAIChatConfig` 节点
2. 填入你的 OpenAI API Key（格式：`sk-xxxxx`）
3. Model 选择 `gpt-4o`

### 第 2 步：输入小说文本

1. 在 `小说原文输入` 节点中粘贴你的小说章节
2. 设置 `主要角色数` 和 `分镜数量`

### 第 3 步：运行分析

1. 点击 `Queue Prompt` 按钮
2. 等待 OpenAI 分析完成（约 30-60 秒）
3. 查看人物/场景/动作分析结果

### 第 4 步：生成角色板

1. 分析完成后，英文提示词会自动填充
2. RecraftV4 节点会根据提示词生成角色三视图
3. 使用 `PreviewAny` 节点查看结果

### 第 5 步：生成视频

1. 将角色图连接至 `WanImageToVideoApi`
2. 设置视频参数（帧数、分辨率等）
3. 使用 `PreviewAny` 查看视频预览

## 💡 关键节点说明

| 节点类型 | 用途 | 关键参数 |
|---------|------|---------|
| OpenAIChatConfig | AI 文本分析 | API Key, Model, Temperature |
| RecraftV4TextToImageNode | 角色图生成 | Style, Resolution, Steps |
| IdeogramV3 | 场景图生成 | Quality, Resolution |
| ByteDanceImageNode | 动作参考图 | Model, Seed |
| WanImageToVideoApi | 人物视频 | Duration, Resolution, CFG |
| LumaVideoNode | 视频合成 | Model, Frames |
| SAM3_VideoTrack | 角色追踪 | Prompts |
| ControlNetApplyAdvanced | 一致性控制 | Strength, Scale |

## 🎯 确保一致性的技巧

### 1. 角色一致性
- 使用相同的 Seed 值生成多视角
- PhotoMaker 节点锁定面部特征
- ControlNet Union 保持姿势一致

### 2. 分镜连贯性
- SAM3 跨镜头角色追踪
- ControlNet 动作链验证
- 统一的场景色调和光影

### 3. 视频质量
- 先用 576x576 测试，确认效果后再用 720x720
- 使用 LTX-2.3-22B 模型进行视频超分
- 添加 Frame Interpolation 提升流畅度
