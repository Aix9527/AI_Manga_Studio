# 视频生成提示词指南 (Seedance2 Skill)

## 镜头语言速查
| 术语 | 说明 |
|------|------|
| Push in / Slow push | 镜头推向主体 |
| Pull back | 镜头拉远 |
| Pan left/right | 水平摇镜 |
| Tilt up/down | 垂直摇镜 |
| Track / Follow shot | 跟随主体移动 |
| Orbit / Revolve | 环绕主体 |
| One-take / Oner | 无剪切长镜头 |
| Hitchcock zoom | 推拉变焦(眩晕效果) |
| First-person POV | 第一人称视角 |
| Close-up | 特写(面部) |
| Wide / Establishing shot | 广角/定场镜头 |
| Low angle | 仰拍 |
| Bird's eye | 俯拍 |

## 漫剧分段时间轴格式
0s-3s: [开场场景描述，镜头运动]
3s-6s: [中间发展]
6s-9s: [高潮或关键动作]
9s-12s: [收尾，最终画面]

## 漫剧视频提示词模板
[Subject/Character Setup] + [Scene/Environment] + [Action/Motion] +
[Camera Movement] + [Timing Breakdown] + [Transitions/Effects] +
[Audio/Sound Design] + [Style/Mood]

## 常用风格修饰
- 电影质感: Cinematic quality, film grain, shallow depth of field
- 动漫风格: Anime style, vibrant colors, clean lineart
- 氛围: Tense and suspenseful / Warm and healing / Epic and grand
