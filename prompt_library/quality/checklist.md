# 提示词质量检查清单 (Prompt Master 37 Anti-Patterns)

## 生成前必查
- [ ] 指定了明确的艺术风格 (anime/manga/realistic)
- [ ] 包含 Negative Prompt (不含: 模糊/水印/畸形/低质量)
- [ ] 指定了分辨率和采样器参数
- [ ] 角色提示词包含: 名称+性别+外貌+服装
- [ ] 场景提示词包含: 描述+时间+天气+氛围
- [ ] 视频提示词包含: 镜头运动+时间轴分段

## 常见反模式速查
| # | 反模式 | 修复 |
|---|--------|------|
| 1 | 任务描述模糊 | 精确到角色名、场景号、镜头号 |
| 2 | 缺少成功标准 | 明确分辨率、帧数、时长目标 |
| 3 | 忽略负面提示 | 必须包含 Negative Prompt |
| 4 | 风格冲突 | 检查 anime vs realistic 一致性 |
| 5 | 角色特征矛盾 | 同一角色跨镜头保持外貌一致 |
| 6 | 遗漏镜头参数 | 补充 FPS、时长、转场方式 |
